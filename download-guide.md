## INTRODUCTION
Windows XP supports two common installer types:

- **MSI** – Microsoft Installer Package  
- **EXE** – Executable Installer  

This guide explains how to safely install both on Windows XP.

----------------------------------------------------

## BEFORE YOU BEGIN
✔ Log in as an **Administrator**  
✔ Close all running programs  
✔ Scan installer files with antivirus (if available)  
✔ Create a **System Restore Point** (recommended)

----------------------------------------------------

## INSTALLING MSI FILES (.MSI)

### METHOD 1: STANDARD INSTALL
1. Locate the `.msi` file
2. Double-click the file
3. Windows Installer will start
4. Click **Next**
5. Accept the license agreement
6. Choose install location (default recommended)
7. Click **Install**
8. Click **Finish** when complete

----------------------------------------------------

### METHOD 2: USING RUN COMMAND
1. Click **Start → Run**
2. Type:
3. Press **ENTER**
4. Follow the on-screen instructions

----------------------------------------------------

### METHOD 3: SILENT INSTALL (ADVANCED)
Used mainly by system administrators.


⚠ No prompts will appear during installation.

----------------------------------------------------

## INSTALLING EXE FILES (.EXE)

### STANDARD INSTALL
1. Locate the `.exe` installer
2. Double-click the file
3. If prompted, click **Run**
4. The Setup Wizard will open
5. Click **Next**
6. Choose desired options
7. Click **Install**
8. Click **Finish**

----------------------------------------------------

### RUNNING AS ADMINISTRATOR
If installation fails:

1. Right-click the `.exe` file
2. Select **Run as...**
3. Choose **Administrator**
4. Enter the administrator password if prompted

----------------------------------------------------

## COMMON INSTALL OPTIONS
You may be asked to configure:
- License Agreement
- Destination Folder
- Start Menu Folder
- Desktop Shortcut
- Optional Components

----------------------------------------------------

## TROUBLESHOOTING

**Installer does not start**
- Re-download the file
- Verify the file extension
- Temporarily disable antivirus

**MSI Error 1603**
- Confirm administrator rights
- Check available disk space
- Restart Windows XP and try again

**"Not a valid Win32 application"**
- Program may require a newer version of Windows
- Installer file may be corrupted

----------------------------------------------------

## UNINSTALLING PROGRAMS
1. Click **Start → Control Panel**
2. Open **Add or Remove Programs**
3. Select the program
4. Click **Remove**
5. Follow the uninstall instructions

----------------------------------------------------

## SECURITY NOTICE
⚠ Windows XP is no longer secure.

- Install software from **trusted sources only**
- Avoid unknown or modern installers
- Do not connect to the internet when possible

----------------------------------------------------

## FINAL NOTES
- **MSI** installers provide cleaner, managed installs
- **EXE** installers vary depending on the developer

Always read each prompt carefully.

**"Compatible with Windows XP" ✔**
